Diablo 2 Runewords

This website is an API for nightbot commands. We play two mods: Path of Diablo and Project Diablo 2.

API Docs:

  • Path of Diablo Runewords: /api/podrw/:name
    For example: /api/podrw/Infinity
  • Project Diablo 2 Runewords: /api/pd2rw/:name
    For example: /api/pd2rw/Infinity
  • Original D2 runewords can be fetch at: /api/d2runewords/:name
    For example: /api/d2rw/Infinity
  • You can also search for Runes:
    /api/runes/:rune
    For example:
    /api/runes/Ral
  • Or Items:
    /api/items/:item
    For example:
    /api/items/Sword
  • See the full list of Runewords:
    /api/
