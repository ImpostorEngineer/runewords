API Routes

  • /api/d2rw returns the full D2R runeword list.
  • /api/d2rw/name/:name searches by runeword name.
  • /api/d2rw/runes/:rune searches by rune name.
  • /api/d2rw/item/:item searches by allowed item base.
  • /api/runes/:rune and /api/items/:item remain available as shortcuts.
